Overview

Referencia de la API (0.0.1)

Section titled “Referencia de la API (0.0.1)”

BluStore API es un backend REST simple para manejar usuarios y productos.

  • Auth: Registro y login con JWT.

  • Productos: CRUD completo (crear, leer, actualizar, eliminar).

  • Seguridad: Endpoints protegidos con Bearer Token.

  • Otros: Health check del servicio.

Todo usa JSON siguiendo un contrato formal de respuesta.

Primeros pasos (Postman):

  • Configurar baseURL en: https://blustore-api.vercel.app o tu localhost si es que no aparece disponible

  • Login Demo (Prueba de JWT) [token automatizado]

  • Register -> Login (Variables automaticas) [token automatizado]

  • POST, PATCH, DELETE (Necesitan token) [ejecutar Register/Login antes]

  • OpenAPI version: 3.0.0
